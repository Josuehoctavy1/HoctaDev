# Description du projet
Ce portfolio a été conçu pour présenter mon travail en tant que développeur web, avec un design moderne et épuré. Il permet aux visiteurs de découvrir mes compétences techniques, mes projets réalisés et d'entrer en contact avec moi facilement. Le site utilise des technologies web modernes telles que HTML, CSS, et JavaScript.

# Fonctionnalités
Page d'accueil : Présentation de mon profil avec un lien vers mon CV et mes réseaux sociaux professionnels (LinkedIn, GitHub).

À propos de moi : Section détaillant mon parcours, mes compétences techniques et une photo professionnelle.

Mes projets : Galerie des projets réalisés avec des liens vers leur code source sur GitHub ou leurs démos en ligne.

Compétences : Liste de mes compétences techniques avec des icônes et graphiques de progression.

Contact : Formulaire pour envoyer un message directement via mon email professionnel.

Technologies utilisées
Ce projet utilise les technologies suivantes :

HTML : Structure du contenu du site.

CSS : Mise en page et design, avec Flexbox pour la disposition et des animations pour améliorer l'interaction.

JavaScript : Fonctionnalités interactives, notamment pour changer le thème entre clair et sombre.

LocalStorage : Pour mémoriser le thème sélectionné par l'utilisateur.

Comic Sans MS : Police utilisée pour donner un look décontracté et personnel au site.

Personnalisation
Voici quelques éléments que vous pouvez personnaliser pour adapter ce portfolio à vos besoins :

Votre photo/avatar : Remplacez img/ton_avatar.jpg par votre propre photo ou avatar.

Projets : Modifiez les sections de projets dans index.html pour ajouter vos propres réalisations, avec des liens vers GitHub ou des démos en ligne.

Compétences : Mettez à jour la liste des compétences avec vos technologies et outils favoris.

CV : Remplacez le lien vers le fichier assets/CV.pdf par votre propre CV.

Réseaux sociaux : Mettez à jour les liens vers vos profils professionnels LinkedIn, GitHub, etc.
